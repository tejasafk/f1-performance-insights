# f1-performance-insights
A clean Python pipeline using Pandas and NumPy to clean messy Formula 1 race results. It converts raw lap timestamps into workable durations, parses driver retirement reasons into structured tracking columns, and maps starting positions against final race finishes using Seaborn charts.
